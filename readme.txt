= Checkout - Twenty Fourteen =

This is a child theme folder for the Shopping Cart plugin.

== Installation ==

* Obtain the ZIP from Github: https://github.com/tribulant/checkout-theme-twentyfourteen
* Extract the ZIP file to obtain the “checkout-theme-twentyfourteen” folder on your computer. Rename the folder to “checkout”
* Access your WordPress theme folder via FTP “wp-content/themes/*theme*/“ where *theme* is the theme’s folder name.
* Upload the “checkout” folder to the folder of your WordPress theme or child theme.
